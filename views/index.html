Metric/Imperial Converter

Example usage

fill valid input in the box or you can just add this to url

/api/convert?input=*YourQuery*

Front-End

some valid inputs:

replace *YourQuery* with

4gal, 1/2km, 4.5/5km,9.7mi,kg,lbs,mi,gal,l,km

Example return

{ initNum: 3.1, initUnit: 'mi', returnNum: 4.98895, returnUnit: 'km', string: '3.1 miles converts to 4.98895 kilometers' }
